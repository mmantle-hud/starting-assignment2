# Starting Assignment 2
## If you are using Codespaces

From your Assignment 1 Github page 
- Select 'settings'
- Select 'Template repository' to make your repository a template repository. 

Back on the hompepage of your repository
- Select 'Use this template' and 'Create a new repository'.
- Specify a repository name e.g. Assignment 2.
- Make the visibility 'private'.
- Select 'Create repository'

Now go to the Codespaces homepage - https://github.com/codespaces
- Select 'New codespace' 
- From the repository option select your new Assignment 2 respository
- Select Create Codespace.
    - You may need to authorise the creation of the codespace. Agree to this. 

Your Assignment 2 codespace should start

- Next, follow the instructions from https://github.com/CHT2520-web-prog/intro-to-laravel/blob/main/intro_to_Laravel.md#install-dependencies to install the dependencies and configure the application.

- Run `php artisan serve` to check your application works.
- Make a change to the README e.g. Remove the contents and enter

```
# Assignment 2
```
- Push your changes to the Github repo and make sure you can see the changes in the Assignment 2 Github repsoitory.
- If you have any problems ask for help ASAP so you can move on with Assignment 2.

## If you are using XAMPP
- From https://github.com/ create a new repository
- Give it a name e.g. Assignment2
- Make sure it is private.

From your local copy of Assignment 1. Enter the following git command to change the remote to your Assignment 2 repository (make sure you replace 'name-of-assignment2-repo' with the actual name of your repository. 
```
git remote set-url origin https://github.com/name-of-assignment2-repo
```
- Make a change to the README e.g. Remove the contents and enter

```
# Assignment 2
```
- Push your changes to the Github repo and make sure this also works.
- If you have any problems ask for help ASAP so you can move on with Assignment 2.
